создаем приложение npx create-react-app .     Точка указана для установки нашего приложения в данной папке в которой находимся 

npm run start => Запускает сервер разработки.

npm run build => Объединяет приложение в статические файлы для производства.

npm run eject => Удаляет этот инструмент и копирует зависимости сборки, файлы конфигурации  и скрипты в каталог приложения. Если ты это сделаешь, ты не сможешь вернуться!

npm run test => Запускает тестовый бегун. 


index.js => ТОчка входа нашего приложения 

Удаляем все кроме ( public=> index.html / favicon.ico)  (src => index.css / index.js)
